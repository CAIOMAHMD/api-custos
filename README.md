# Azure Cost Dashboard

Dashboard simples e direto para visualizar custos do Azure usando **FastAPI** no backend e **Streamlit** no frontend.  
As TAGs são processadas dinamicamente e exibidas de forma consolidada.

---

## 🛠 Pré‑requisitos

Antes de iniciar, instale:

### ✔ Python 3.10 ou superior  
Verifique com:

```bash
python3 --version

---

## 🚀 Como iniciar

Use o script `launch.sh` para subir tudo de uma vez:

```bash
./launch.sh
